# bettersocial-test-purbo
bettersocial test
frontend web dan backend menggunakan laravel
Breeze & Vue

## Petunjuk dan langkah-langkah:
* composer require laravel/breeze --dev
* php artisan breeze:install vue
 
* php artisan migrate
* npm install
* npm run dev

## Jalankan client app
php artisan serve
